GIT

  • How to create a new branch

    • Navigate to main directory
    • git checkout main (select main branch)
    • git checkout -b feature/newbranchname
    • git status (to confirm)

  • How to commit new branch

    • git status (confirm branch)
    • git add -A (add all changes to staging area)
    • git commit -m "commit comment" (commit staged changes)
    • git pull origin main (sync with main)
    • git push origin feature/branch-name (push request)

  • How to merge branch to main

    • Go to repo on github
    • Select "Compare & pull request"
    • Give request title and submit (ensure feature is merging to correct place)
    • Merge

GIT

  • How to create a new branch

    1. Navigate to main directory
    2. git checkout main (select main branch)
    3. git checkout -b feature/newbranchname
    4. git status (to confirm)

  • How to commit new branch

    1. git status (confirm branch)
    2. git add -A (add all changes to staging area)
    3. git commit -m "commit comment" (commit staged changes)
    4. git pull origin main (sync with main)
    5. git push origin feature/branch-name (push request)

  • How to merge branch to main

    1. Go to repo on github
    2. Select "Compare & pull request"
    3. Give request title and submit (ensure feature is merging to correct place)
    4. Merge
